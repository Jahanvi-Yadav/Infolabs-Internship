# 🏠 RentEase - Rental Property & PG Management System

RentEase is a Django-based web application developed to simplify rental property and PG management. The system allows tenants to browse properties, book rentals, make payments, manage wishlists, and submit reviews through an intuitive interface.

---

## 📌 Features

### Tenant Module

- Tenant Registration
- Tenant Login & Logout
- Session Authentication
- Profile Management

### Property Module

- View Available Properties
- Property Details
- Property Images
- Property Categories
- Property Status

### Booking Module

- Book Rental Property
- Check-in & Check-out
- Booking History

### Payment Module

- Online Payment Simulation
- Payment Status
- Transaction ID Generation

### Wishlist Module

- Add Property to Wishlist
- View Wishlist

### Review Module

- Property Ratings
- Tenant Reviews

### Contact Module

- Contact Form
- Store Customer Queries

### Admin Panel

- Manage Properties
- Manage Owners
- Manage Tenants
- Manage Bookings
- Manage Payments
- Manage Reviews

# Installation

Clone Repository

```bash
git clone https://github.com/yourusername/RentEase.git
```

Open Project

```bash
cd RentEase
```

Install Requirements

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

Run Server

```bash
python manage.py runserver
```

Visit

```
http://127.0.0.1:8000/
```
